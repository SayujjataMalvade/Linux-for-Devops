# ⚙️ Process Management in Linux

## Viewing Running Processes

### ps -ef
Displays all running processes.

```bash
ps -ef
```

### Important Fields:
- **PID** – Process ID
- **PPID** – Parent Process ID
- **CMD** – Command executing the process

---

## Real-Time Performance Monitoring

### top
Displays live system performance.

```bash
top
```

### Shows:
- CPU utilization
- Memory usage
- Load average
- Active processes
